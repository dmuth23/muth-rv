# Archive

Research from earlier phases of the search, kept because good work shouldn't vanish when a
decision moves on. Everything here was accurate when written and is **no longer maintained** —
each page carries a banner saying why it was superseded.

The big shift that created this archive: in July 2026 the mission was re-centered on
**national- and state-park camping**, where campground length limits punish large rigs. That
pivoted the search from a 30–35' gas Class A to a **Class C**, and retired the Class-A-specific
research (the gas Class A model landscape, the Ford F-53 chassis math, the diesel-pusher
primer). The school-calendar travel analysis was retired separately — the kids are
homeschooled, so it solved a problem this family doesn't have.
