# The Muth Family RV Docket

Research docket for buying a **used, gas-powered Class A motorhome (30–35 ft)** for a
family of four (two adults, two 4-year-olds), with full-time remote work aboard,
mostly-campground use plus real boondocking capability, and room for ~6 bikes.
Purchase window: late 2026 through 2028.

**📖 Read the docket as a website: https://dmuth23.github.io/muth-rv/**

Or browse the markdown directly in [`docs/`](docs/).

## What's in here

The docket covers: RV classes 101 and the gas/used decisions, the used gas Class A
model landscape, the Ford F-53 chassis and weight/payload math, how to buy used
(inspection, negotiation, timing), RVing with young kids (including child-seat
safety), working remote from the rig (Starlink, cellular, dual-monitor workspace),
boondocking power upgrades, carrying six bikes, must-have gear and upgrades, true
ownership costs (with New Jersey specifics), trip planning and driving, and the
classic beginner mistakes with operating checklists.

Every doc ends with a curated "Read more" list — those links are the family
reading list.

## How this was built

Compiled by Claude Code via a fan-out of web-research agents, with an adversarial
fact-check pass on high-stakes claims (safety, legal, money). It's research, not
gospel: **verify current prices, specs, and laws before acting on anything here.**
